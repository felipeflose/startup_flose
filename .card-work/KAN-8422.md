# KAN-8422 — Backend Service para Integração Jira & Git com Utilitários de Slugificação

**✍️ Criado por:** Bruno Coord RH (Coordenador de Recrutamento & Seleção)
**👤 Responsável:** Helena Talent
**📁 Arquivo:** `server.cjs`

## Análise PO (Gemma 4)
Serviço de backend construído em Node.js/Express, focado na automação de fluxos de trabalho entre sistemas de gestão (Jira) e controle de versão (Git). O módulo implementa funções robustas de manipulação de strings (slugificação avançada para títulos e resumos) e a capacidade de executar comandos Git diretamente via `child_process`. Ideal para automatizar a criação de branches, nomes de tickets e preparação de dados para sistemas externos.
