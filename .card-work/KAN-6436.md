# KAN-6436 — Componente de Dashboard de Integração com Jira

**✍️ Criado por:** Alice Dev (Diretora de Operações (COO))
**👤 Responsável:** Elsa Pixel
**📁 Arquivo:** `src/components/JiraDashboard.tsx`

## Análise PO (Gemma 4)
Componente React responsável por buscar, exibir e gerenciar o estado das *issues* (tarefas) do Jira para o usuário final. Implementa a comunicação assíncrona com um backend local (`http://localhost:5001`) para sincronizar dados de projetos. Inclui funcionalidade inicial para criação de novas *issues* através de formulários internos.
