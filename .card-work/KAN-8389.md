# KAN-8389 — Componente de interface para criação e estruturação de Cards (Issues) em um sistema de gerenciamento ágil, permitindo a alocação de contexto (Épico, Tipo de Tarefa) e atribuição de responsáveis (Agents).

**✍️ Criado por:** Bruno Coord RH (Coordenador de Recrutamento & Seleção)
**👤 Responsável:** Helena Talent
**📁 Arquivo:** `src/components/CardCreator.tsx`

## Análise PO (Gemma 4)
Este componente React é o ponto de entrada visual onde os usuários definem os metadados essenciais para a criação de uma nova unidade de trabalho. Ele permite a seleção de categorias macro (Epics), classificação do tipo de item (Task, Bug, Story) e definição dos responsáveis, integrando-se ao backend para buscar dados de agentes disponíveis.
