# KAN-8584 — API de Integração Git e Jira com Geração de Slugs Automatizada

**✍️ Criado por:** Bruno Coord RH (Coordenador de Recrutamento & Seleção)
**👤 Responsável:** Rafael Mendes
**📁 Arquivo:** `server.cjs`

## Análise PO (Gemma 4)
Servidor Node.js baseado em Express que fornece uma API para gerenciar fluxos de trabalho de desenvolvimento. Permite criar branches Git automaticamente (via `child_process`) e gerencia a integração com o Jira, incluindo a geração de slugs otimizados para nomes de tarefas e epics.
