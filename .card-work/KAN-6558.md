# KAN-6558 — Serviço de Backend para Automação de Fluxos de Trabalho Git e Preparação de Dados para Repositórios

**✍️ Criado por:** Alice Dev (Diretora de Operações (COO))
**👤 Responsável:** Arthur GameDev
**📁 Arquivo:** `server.cjs`

## Análise PO (Gemma 4)
Este serviço Node.js implementa uma API Express que atua como um middleware de integração entre a aplicação principal e o sistema de controle de versão (Git), focado em automação de fluxos de trabalho de desenvolvimento e geração de slugs otimizados para gerenciamento de conteúdo. As funcionalidades chave incluem: 

1. **Slugification Avançada:** Funções customizadas (`slugify` e `slugifySummaryShort`) que garantem a conversão de textos complexos em identificadores URL-friendly, essenciais para nomes de branches e arquivos.
2. **Gerenciamento de Branches (Git Integration):** Uma função assíncrona robusta (`createGitBranch`) que executa comandos nativos do Git via `child_process`, permitindo a criação e o checkout de branches diretamente no repositório, garantindo a consistência com as demandas do Jira.
3. **Segurança:** Utilização de variáveis de ambiente para armazenamento seguro das credenciais do Jira (Host, Usuário, Token).
