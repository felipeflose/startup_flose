# KAN-8334 — Vulnerabilidade de Injeção de Comando via `child_process.exec` na função `createGitBranch`

## Análise do Motor Gemma 4
A função `createGitBranch` utiliza `child_process.exec` para executar comandos Git (`git checkout -b ${branchName}`). Se o valor da variável `branchName` for fornecido por uma entrada externa (como um parâmetro de requisição HTTP), existe um risco crítico de Injeção de Comando (Command Injection). Um atacante poderia injetar comandos arbitrários no sistema operacional, comprometendo a execução do servidor e potencialmente levando à escalada de privilégios ou exclusão de dados. Critério de aceitação: Implementar validação rigorosa da entrada e utilizar métodos mais seguros para execução de comandos externos.
