## Análise PO (Gemma 4)
Componente React/TypeScript responsável por buscar, exibir e gerenciar dados de issues do Jira em uma interface de usuário. Implementa a lógica de busca de dados via API local (simulando integração com o Jira) e fornece funcionalidades básicas de seleção de issues.
