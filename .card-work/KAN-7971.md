# KAN-7971 — Dashboard de Integração com Jira e Criação de Tickets

**✍️ Criado por:** Bob Delivery (Gerente de Engenharia)
**👤 Responsável:** Lucas Cloud
**📁 Arquivo:** `src/components/JiraDashboard.tsx`

## Análise PO (Gemma 4)
Componente React que atua como o ponto de entrada visual para os dados do Jira, permitindo aos usuários visualizar issues abertos, selecionar tickets específicos (via `onSelectIssue`) e fornecer uma interface para a criação de novos tickets diretamente na aplicação Flose. Responsável pela comunicação com o backend local para a recuperação dos dados.
