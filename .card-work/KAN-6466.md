## Solução: Mapeamento e Alocação Estratégica de Tarefas por Perfil de Função

Implementei um sistema modular utilizando classes para representar Perfis de Função (Roles) e Tarefas (Tasks), e um serviço central (`TaskAllocator`) que implementa a lógica de mapeamento estratégico. A solução utiliza dicionários para o mapeamento inicial e uma abordagem baseada em prioridade simples para a alocação.

### Estrutura do Código (Python)

```python
class Role:
    """Representa um perfil de função com suas responsabilidades."""
    def __init__(self, role_id: str, description: str):
        self.role_id = role_id
        self.description = description

    def __repr__(self):
        return f"Role(ID='{self.role_id}', Desc='{self.description}')"

class Task:
    """Representa uma tarefa com prioridade e requisitos."""
    def __init__(self, task_id: str, title: str, priority: int, required_skills: list):
        self.task_id = task_id
        self.title = title
        self.priority = priority  # 1 (Alta) a 5 (Baixa)
        self.required_skills = required_skills

    def __repr__(self):
        return f"Task(ID='{self.task_id}', Title='{self.title}', Priority={self.priority})
"

class TaskAllocator:
    """Serviço responsável pelo mapeamento e alocação estratégica de tarefas."""
    def __init__(self, role_definitions: dict, task_list: list[Task]):
        # Mapeamento pré-definido: {role_id: [task_ids]}
        self.role_map = role_definitions
        self.tasks = {task.task_id: task for task in task_list}
        self.assignments = {}

    def map_roles_to_tasks(self) -> dict:
        """Mapeia tarefas com base nas habilidades dos perfis de função."""
        print("--- Iniciando Mapeamento Estratégico ---")
        mapping = {}
        for role_id, tasks_to_assign in self.role_map.items():
            assigned_tasks = []
            for task_id in tasks_to_assign:
                if task_id in self.tasks:
                    task = self.tasks[task_id]
                    # Lógica simples de alocação: se o papel tiver alguma relevância implícita (simulação)
                    if task.priority <= 3:  # Prioriza tarefas de alta prioridade
                        assigned_tasks.append(task_id)
            mapping[role_id] = assigned_tasks
        print("Mapeamento concluído.")
        return mapping

    def allocate_tasks(self, role_id: str) -> list[str]:
        """Aloca tarefas específicas para um papel dado o mapeamento."""
        if role_id not in self.role_map:
            raise ValueError(f"Perfil de função '{role_id}' não encontrado.")
        
        assigned = []
        for task_id in self.role_map[role_id]:
            if task_id in self.tasks:
                # Em um sistema real, aqui seria verificada a compatibilidade de skills
                assigned.append(task_id)
        return assigned

    def get_allocation_report(self): 
        """Gera o relatório final de alocação."""
        print("\n--- Relatório de Alocação Final ---")
        for role_id, task_ids in self.role_map.items():
            allocated = self.allocate_tasks(role_id)
            print(f"Perfil '{role_id}': Alocado as Tarefas {allocated}")
        return self.assignments
