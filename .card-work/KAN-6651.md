# KAN-6651 — API de Automação de Fluxo de Trabalho Git e Conteúdo

**👤 Responsável:** David Dev
**📁 Arquivo:** `server.cjs`

## Análise PO (Gemma 4)
Este é o backend Node.js/Express responsável por fornecer endpoints para automação de tarefas de desenvolvimento (Git) e manipulação de strings (slugificação). Ele atua como uma camada de serviço que integra a lógica de nomenclatura de conteúdo com comandos do sistema operacional, permitindo que aplicações front-end interajam com repositórios Git e sistemas de gerenciamento de projetos (como Jira). As funções chave incluem: 
1. **Slugification:** Criação de slugs amigáveis para URLs e nomes de arquivos.
2. **Gerenciamento de Branches:** Execução de comandos `git checkout -b` via `child_process` para criar branches automaticamente.
3. **Integração de Credenciais:** Leitura segura das credenciais do Jira para futuras integrações.
