**✍️ Criado por:** Felipe Flose (CEO (Chief Executive Officer))
**👤 Responsável:** Lucas Augusto Silva
**📁 Arquivo:** `server.cjs`

## Análise PO (Gemma 4)
Este módulo Node.js atua como uma camada robusta de orquestração, fundamental para a integração entre nossos processos internos de desenvolvimento e sistemas externos de gestão de projetos (Ex: Jira). Ele vai além de um simples servidor Express, pois encapsula lógica crítica de ciclo de vida do software (SDLC).

**Funcionalidades Chave:**
1. **Controle Git Resiliente:** Implementa uma rotina avançada de *branch management* (`createGitBranch`) que tenta primeiro criar o branch e, em caso de falha esperada (já existir), executa um fallback inteligente para checkout no branch existente. Isso garante a estabilidade do pipeline de deploy.
2. **Sanitização Inteligente de Dados:** Oferece utilitários `slugify` e `slugifySummaryShort`, que garantem que todo texto consumido ou gerado para URLs, slugs e títulos sejam universais (sem acentos, caracteres especiais) e concisos, evitando erros em APIs externas.
3. **Gateway de Integração Externa:** Prepara a infraestrutura para consumo seguro de variáveis de ambiente sensíveis (`JIRA_HOST`, `JIRA_TOKEN`) e utiliza pacotes como Axios, posicionando este serviço como o *gateway* oficial para interações com sistemas CRM/PM, minimizando dependências diretas nos clientes.
