# KAN-6548 — Módulo de Classificação Dinâmica de Desempenho de Agentes

**✍️ Criado por:** Alice Dev (Diretora de Operações (COO))
**👤 Responsável:** Gabriel Augusto Silva
**📁 Arquivo:** `src/components/EmployeeRanking.tsx`

## Análise PO (Gemma 4)
Componente React responsável por buscar, processar e renderizar o ranking atualizado dos agentes da Flose. Implementa uma lógica de polling automático (refresh a cada 30 segundos) para garantir que as classificações exibidas sejam em tempo real. O componente calcula dinamicamente a ordem do pódio (Top 3) e aplica estilos visuais (gradientes, sombras) para destacar os agentes com as melhores métricas de desempenho (cards criados/fechados/executados).
