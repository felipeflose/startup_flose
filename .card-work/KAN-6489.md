# KAN-6489 — Desenvolvimento do Componente de Dashboard Integrado com Jira

**✍️ Criado por:** Rafael Augusto Silva (Product Owner Sênior)
**👤 Responsável:** Elsa Pixel
**📁 Arquivo:** `src/components/JiraDashboard.tsx`

## Análise PO (Gemma 4)
Implementar o componente React `JiraDashboard` responsável por buscar, exibir e gerenciar visualmente as issues e commits dos projetos no painel. O foco desta implementação é garantir a correta integração com a API local (`http://localhost:5001/api/jira/issues`), gerenciamento de estados assíncronos (loading, erro) e a funcionalidade básica de criação de novas issues via formulário. Necessita da definição robusta das interfaces de dados para mapear corretamente os campos complexos do Jira (descrições aninhadas e tipos de issue).
