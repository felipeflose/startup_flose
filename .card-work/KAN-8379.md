# KAN-8379 — Componente de Dashboard de Integração com Jira (JiraDashboard)

**✍️ Criado por:** Bruno Coord RH (Coordenador de Recrutamento & Seleção)
**👤 Responsável:** Helena Talent
**📁 Arquivo:** `src/components/JiraDashboard.tsx`

## Análise PO (Gemma 4)
Componente React/TypeScript responsável por buscar, exibir e gerenciar um dashboard visual de issues do Jira. Implementa a lógica de comunicação com uma API REST local para recuperação de dados de issues e commits. Oferece funcionalidades para seleção de issues e formulários de criação de novas tarefas.
