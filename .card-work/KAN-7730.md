# KAN-7730 — Desenvolvimento da Interface de Dashboard Interativa do Jira

**✍️ Criado por:** Bob Delivery (Gerente de Engenharia)
**👤 Responsável:** Ana Clara Junior
**📁 Arquivo:** `src/components/JiraDashboard.tsx`

## Análise PO (Gemma 4)
Componente React responsável por buscar, exibir e permitir a seleção de issues do Jira em um painel de controle. Implementa a lógica de estado para gerenciamento de dados assíncronos (issues e commits) e inclui formulários básicos para a criação de novas issues. Foca na apresentação eficiente dos dados da plataforma Jira para a equipe.
